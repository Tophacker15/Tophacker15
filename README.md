### 🚀 Featured Builds

**🔗 Pi Network / Stellar Bot Infrastructure**
Self-hosted Horizon node + a full automation stack for Pi Network and Stellar — VPS-deployed, running live bot fleets against production blockchain infra.
`Python` `Stellar SDK` `Horizon` `VPS/Linux`

**📱 TopSMS — Virtual Number Rental Platform**
SMS number rental service for the Nigerian market — US numbers, Naira payments, built for scale.
`Node.js` `Payments` `REST API`

**🔢 NumVault — SMS Number Rental SaaS**
Multi-provider number rental SaaS with Supabase Auth, 5sim integration, and crypto top-up flows.
`Supabase` `5sim API` `Crypto Payments`

**📈 Algorithmic Trading Bots**
Forex/algo trading systems built on NautilusTrader, ib_insync, and fxcmpy, plus a custom EA31337 fork for MT4/5 strategy automation.
`NautilusTrader` `Interactive Brokers` `FXCM` `MQL`

Multi-session WhatsApp bot on Baileys with 90+ commands — from utilities to automation workflows.
`Baileys` `Node.js` `Multi-session`
